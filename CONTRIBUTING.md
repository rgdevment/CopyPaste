# Contributing to CopyPaste

First off, thank you for considering contributing to CopyPaste! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

CopyPaste is an open-source project maintained by **rgdevment**. We believe in high-performance tools and privacy respecting software.

## 🛠 How Can I Contribute?

### Reporting Bugs
- Ensure the bug was not already reported by searching on GitHub under [Issues](https://github.com/rgdevment/CopyPaste/issues).
- If you're unable to find an open issue addressing the problem, open a new one. Be sure to include a **title and clear description**, as much relevant information as possible, and a code sample or an executable test case demonstrating the expected behavior that is not occurring.

### Pull Requests
1.  Create a branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code follows the existing style (clean, minimalist, strictly commented in English).
5.  Open a Pull Request pointing to the `main` branch.

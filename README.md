# CopyPaste

An open-source clipboard manager for Windows.

## About the Project
CopyPaste is an open-source clipboard manager designed with an obsessive focus on stability and a minimalist aesthetic inspired by the macOS experience. Currently, there is nothing quite like it for Windows.

While there are functional and wonderful "tanks" out there, we are looking for something else. This project is born from motivation, will, and a specific necessity: a tool that stays out of the way, performs reliably, and feels right.

## Why are we building this?
* **Minimalist Aesthetics**: Inspired by the clean, card-based interface seen in macOS utilities.
* **Rock-Solid Stability**: Focused on being a dependable tool for daily work.
* **Modern Stack**: Built using C# 14 and .NET 10 to explore the latest performance improvements.
* **Resource Efficiency**: Aiming for low overhead by utilizing NativeAOT and disk-based persistence.

## Motivation & Contributions
This project is driven by the desire to create something we actually want to use. We are not trying to impress anyone; we are just building a tool that fits our workflow and hopefully inspires others to contribute or learn along the way.
